I Flexbox

Parece float, mas é flex!

Conheça um pouco mais!

Framework open source para facilitar seus grids em flexbox!

Você pode usar nosso CSS processado ou se preferir pode baixar nosso código fonte e adaptar da melhor maneira para o seu projeto.

Nosso sistema de grid foi pensado em usuários que já trabalham com frameworks que utilizam float. Assim você pode utilizar como sempre usou mas terá algumas vantagens do flexbox em suas mãos.

Todos os componentes foram pensados para funcionar nos mais diversos dispositivos. Você poderá controlar individualmente o comportamento do elemento nos mais diversos dispositivos.
